How to collect, logging and monitor temperature data using raspberry

Raspberry is popular to use in different IoT applications.TMP36, TC74, BME280sensors, and Thingspeak Database , MQTTX , MangoDB servers has been used to store and monitor data.

Here is how IoT works with raspberry in a nuttshell; First, room temperature data has been collected by using mentioned sensors and raspberry pi. Low pass filter was used to collect data. Then, that data stored to the cloud. Data monitoring app has been used to monitor data.
